\"TSKTech Facebook Likes\" widget display facebook like box. The Shortcode allows you to display the FB Like Box anywhere

== Description ==
TSKTech Facebook Like enable Facebook Page owners to attract and gain Likes from their own website.
The Like Box enables users to
-- see page likes,
-- recent posts from the page and
-- which all friends like this page too.

= Options =

1. Facebook Page URL - enter an URL address of your Facebook fan page.
2. Width - set the width of the like box in pixels.
3. Height - Set the height of like box in pixels.
4. Show Faces - choose show or hide faces.
5. Posts - choose show or hide latest posts from your facebook fan page.
6. Cover Photo - choose show or hide Cover Photo.

= Shortcode =

Now Shortcode is available in TSKTech Facebook Like Box and User can easily create shortcode by using Plugin.

[tsk-fblike-box url=\"enter your facebook page url\" width=\"\"	height=\"\" faces=\"true\"	hidecover=\"false\"	tabs=\"\" small_header=\"true\" hide_cta=\"false\" acw=\"true\" locale=\"\"]

== Installation ==
= Option 1 =

1. Upload the entire \"TSKTech Facebook Likes\" folder to your wp-content/plugins/ directory.
2. Activate the plugin through the \'Plugins\' menu in WordPress.

= Option 2 =

1. Download  \"TSKTech Facebook Likes\" to local computer
2. Goto wordpress dashboard > plugin > add new and click on upload.
3. Browse  \"TSKTech Facebook Like\" folder on local computer and click on upload button.
4. Then click on Activate plugin.

== Frequently Asked Questions ==
Is plugin layout responsive?
Yes, TSKTech Facebook Likes plugin is fully responsive. It is built in a way to display all other content properly on any device type and size.
